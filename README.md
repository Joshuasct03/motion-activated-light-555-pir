# Motion Activated Light using PIR Sensor and 555 Timer

# TEAM:
Hashim S N, Joshua Felix, Siddharth S Krishnan, Vibhu V

## Objective
Automatically switch ON a light when motion is detected.

## Working Principle
The PIR sensor detects human motion and triggers a 555 timer configured in monostable mode, activating the load for a fixed duration.

## Concepts Used
- PIR sensing
- 555 timer monostable operation
- Analog timing circuits

## Hardware Components
- PIR Sensor
- NE555 Timer IC
- Relay/LED Load
